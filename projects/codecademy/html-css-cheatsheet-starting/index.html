HTML Table Cheat Sheet

Table Tags

Tag Name Description

<table>

Table

The wrapping element for all table content

<thead>

Table Head

The row(s) defining the column headers

<tbody>

Table Body

The wrapper for the actual data in the table

<tr>

Table Row

A row contains table data cells

<td>

Table Data

Input for data inside the table

<tfoot>

Table Foot

The row(s) containing footer data

Table Attributes

Attribute

Name

Description

<colspan>

Column Span

Defines how many columns a td element should span

<rowspan>

Row Span

Defines how many rows a td element should span
